# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

Build it using ReactJS, CSS, JS, jQuery and AJAX as front-end. Used Node and Express as a back-end.

## Dependencies

- ReactJS
- Express
- Node 5.10.x or above
